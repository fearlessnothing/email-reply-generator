# 📧 Email Reply Generator

## 📌 Project Overview
The Email Reply Generator is a Python-based CLI application designed to help users generate professional email replies quickly and efficiently.  
It uses prompt-based AI generation to produce responses in different tones such as formal, friendly, and short.

---

## 🎯 Objective
The main goal of this project is to:
- Reduce the time spent writing email replies
- Improve communication quality
- Provide structured and tone-aware responses

---

## ⚙️ Features
- Accepts user input (email content)
- Allows tone selection:
  - Formal
  - Friendly
  - Short
- Generates a professional email reply
- Simple command-line interface
- Uses prompt templates for structured output

---

## 🧠 How It Works
1. User inputs the email content
2. User selects the desired tone
3. The system loads a predefined prompt template
4. The prompt is formatted with user input
